Copies a set of files located somewhere on the master node (cf. the Paths are relative to field) into the workspace of the corresponding job on the node performing the build (whether it is a slave node or the master node itself).
This is useful, for example, when some files required for the build to take place are not present in the SCM, but are located in one of the three following locations: HUDSON_HOME/userContent, HUDSON_HOME, or directly in the project's workspace (on the master node).
